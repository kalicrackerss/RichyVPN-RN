🚀 КАК ИСПОЛЬЗОВАТЬ:

ШАГИ 1-3: ЛОКАЛЬНАЯ РАЗРАБОТКА

1. Установить зависимости:
   npm install

2. Установить iOS pods:
   cd ios && pod install && cd ..

3. Запустить (если есть Xcode/Mac):
   npm run ios

ШАГИ 4-6: ОБЛАЧНАЯ СБОРКА IPA (РЕКОМЕНДУЕТСЯ ДЛЯ WINDOWS)

4. Загрузить на GitHub:
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/RichyVPN_RN.git
   git push -u origin main

5. Зарегистрироваться на Codemagic:
   - https://codemagic.io/
   - Sign up with GitHub

6. Подключить репозиторий и запустить сборку:
   - Нажать "Add app"
   - Выбрать RichyVPN_RN
   - Нажать "Set up build"
   - Codemagic автоматически прочитает codemagic.yaml
   - Нажать "Start new build"

РЕЗУЛЬТАТ: скачиваемый IPA файл готов (~15 минут)
